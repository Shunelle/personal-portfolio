VR Interaction / Haptic Interface

Tingle Tennis

SIGGRAPH Asia 2024 Posters

Project Overview

Tingle Tennis 是一個結合 VR 與觸覺回饋（Haptic Feedback）的感官體驗系統。 旨在模擬女性在生理期進行體育活動時的身體不適（經痛、疲累感），透過技術手段打破感官隔閡，促進大眾對生理期挑戰的同理心與理解。

Sensory Simulation

利用經皮神經電刺激 (TENS) 與 Arduino 整合，在遊戲過程中實時模擬經痛的收縮感，其強度會隨著比賽節奏與角色體力值變化。

Game Mechanics

包含「Stamina Bar」與「Blood Absorption Bar」模擬真實生理機制。玩家必須在維持運動表現的同時，應對隨機爆發的經痛干擾。

Technology Stack

Unity (C#) Meta Quest 2 Arduino TENS Device Transformer / GPT (Coach NPC)

My Contributions

  • Project Manager: 負責團隊時程管理、SA 24 論文投稿與 SIGGRAPH Asia 現場展示協調。
  • UX Flow Design: 設計「經痛模擬」與「運動反饋」之間的互動邏輯。
  • HCI Research: 進行 8 名使用者的質性研究，分析沉浸式體驗對「同理心提升」的有效性。
